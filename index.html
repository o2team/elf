elf
灵活可扩展的HTML5多场景应用模板
使用

# Step 1，安装 elf-cli

$ npm i -g elf-cli

# Step 2，创建项目目录

$ mkdir demo && cd demo

# Step 3，执行「elf init -c 案例名」基于指定模板初始化项目，可执行「elf list」查看所有模板，默认 base

$ elf init -c swiper

# Step 4，安装依赖并运行（如果安装 phantomjs 和 node-sass 很慢，可参考 README 设置）

$ npm i && npm start

关于

ELF是凹凸实验室出品的一套面向开发者的灵活可扩展的 HTML5 多场景应用模板，可用来制作各种 HTML5 场景营销活动页面；它基于 Webpack 进行自动化构建，目前共有 4 个组件以及 4 个官方案例模板；利用 ELF 提供的命令行工具 elf-cli，开发者可自由通过模板和组件的组合来快速定制开发 HTML5 场景应用。

特性

  • 微信友好性

    针对微信做了友好性兼容，支持快速配置分享至微信的封面、标题及描述

  • 响应式

    基于REM或Zoom的响应式设计，支持px自动转换REM

  • 样式预处理

    支持Sass、Less或Stylus样式预处理语言及autoprefixer

  • Webpack构建

    基于Webpack进行自动化构建，支持雪碧图合并、资源压缩、hot-reload等特性

组件

  • orientation

    横竖屏重力感应的易用组件

  • preloader

    资源预加载组件

  • swiper

    强大好用的滚屏组件

  • anime

    JS 动效组件
